三个核心设计

🎯 一句话录入

记账 App 留存低的根因是录入摩擦。AI 把「打开→选分类→填金额」压缩成一句话，自动拆多笔、算日期、归分类。

🔍 Agent 全程透明

每轮循环、每次工具调用、每毫秒耗时都可视化。AI 不是黑盒——把它的工作过程做成了产品的一部分。

📏 评测驱动

20 条真实语料评测集（含 4 条坑题），解析准确率 100%；同时作为回归测试，任何 Prompt 改动重跑防止改崩。

技术架构

完整 Agent Harness 二十个环节里选 9 砍 11：Agent Loop（≤6 轮上限）、工具 Dispatch Map、权限管线（增删不对称确认）、Pre/Post Hooks、Skill 热更新（改文件=改产品）、错误恢复三层。每一刀都有理由。

这个项目的另一半价值

产品完成后，我用五条产品化标准（付费习惯 / 10 倍体验差 / 自然分发 / 情绪浓度 / 使用复利）重新评估了记账赛道，主动否掉了它的大众市场方向——支付平台已解决「记录」，AI 解决不了「不想管钱」的动机，大厂「让你花钱」的商业模式与「劝你省钱」的定位存在结构冲突。
